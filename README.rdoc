== README

This is a blog application made using {Jumpstart Labs Blogger Tutorial}[http://tutorials.jumpstartlab.com/projects/blogger.html], as part of {The Odin Project}[http://www.theodinproject.com] curriculum.

Features:
* RESTful interface for blog articles
* adding comments to articles
* tagging articles
* uploading images with paperclip gem
* simple authentication with sorcery gem
* deployed to heroku
